# Plan: YouTube Downloader App
Created: 260319-1529
Status: 🟡 In Progress

## Overview
Desktop application siêu nhẹ dùng để tải video/audio từ YouTube và các trang khác. Tích hợp trực tiếp yt-dlp qua Child Process, được thiết kế không đóng gói kèm dependencies để giữ dung lượng app cực nhỏ. 

## Tech Stack
- Frontend: React + TypeScript + Tailwind CSS (Vite)
- Backend: Go (Wails v2)
- Dependencies: yt-dlp, ffmpeg (Người dùng tự đưa vào PATH)
